# Changelog

## v1.3.0

### 投稿准备

- 新增研究选题池，记录研究问题、文献缺口、创新点、方法、数据来源、优先级与计划节点。
- 新增五维选题评分：创新性、总体可行性、数据条件、方法成熟度和时间可控性。
- 新增草稿准备模块，管理摘要、提纲、作者、字数、图表、参考文献、投稿计划和外部协作链接。
- 新增投稿就绪度诊断，自动识别摘要、作者、主投期刊、正文进度和必需检查项等阻碍。
- 新增投稿前检查清单，并支持添加、标记和删除期刊特有的自定义检查项。
- 新增期刊资料库，支持官网、作者指南、投稿入口、第三方介绍、ISSN、分区、影响因子、收录、OA、APC、审稿周期、接收率和风险信息。
- 新增期刊横向比较，最多并列比较 4 本期刊。
- 支持选题一键建立草稿、草稿一键转入正式投稿管理。

### 数据与部署

- 新增 Supabase 迁移 008：期刊、选题和草稿数据表及 RLS 策略。
- 新增 Supabase 迁移 009：草稿转投稿关联索引。
- 完整备份升级至 v3，同时保存投稿、期刊、选题和草稿，并兼容旧版备份。
- 在线版和离线版共享同一套投稿准备界面与业务逻辑。
- 在线构建、离线单文件构建、原有 Check 工作流及离线隔离检查全部通过。
